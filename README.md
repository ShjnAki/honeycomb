# Rayon de Miel/ Honeycomb

Site développé en **HTML/CSS** pour pratiquer et continuer d'approfondir mes connaissances. 
Les images sont issues de [cinehorizons](https://cinehorizons.net/film/2025) et le background et l'icône ont été générés via **Sora**. 

Déployé sur **IPFS** pour tester l'hébergement décentralisé: 
[accéder au site](https://ipfs.io/ipfs/QmP9YYfTyrGgM8LQ2TZWcW2EtKXLkRyA8FNG5PnBvBVSxe/)

La police utilisée est **Advent Pro** provenant de [Google Fonts](https://fonts.google.com/specimen/Advent+Pro)

## Structure: 

- `index.html` : page d'accueil
- `connexion.html` : page avec formulaires de connexion et inscription
- `global.css` : style communs à toutes les pages
- `style.css` : styles pour éléments spécifiques, pour garder un code organisé et éviter de mélanger les styles globaux et ceux de certaines sections.
- `Media/` : images et assets
